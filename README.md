## How To Use

To clone and run this application, you'll need [Node.js](https://nodejs.org/en/download/) (which comes with [npm](http://npmjs.com)) installed on your computer. From your command line:

```bash

# Install dependencies
$ npm install

# Run app
$ npm start
```
